Platform Capabilities

What the platform does

More than a file uploader — an orchestration layer for the complete genomics workflow, from raw reads to clinical-grade reports.

Data Intake

  • Upload and manage FASTQ and VCF genomic files
  • Organize work by workspace, sample, and analysis record
  • Track analysis state in real time from the web UI

Analysis Workflows

  • FASTQ alignment and variant calling pipelines
  • VCF annotation, filtering, and variant classification
  • hg19 & hg38 reference assembly support

Interpretation Support

  • Review annotated variants and gene-level details
  • Generate structured reports for downstream review
  • AI chatbot for genetics-focused Q&A and assistance

Quality Control

  • Per-sample QC metrics from sequencing runs
  • Pass/fail flags surfaced directly in the UI
  • Available on Standard and Premium plans

Structured Reporting

  • Export analysis results as structured reports
  • Supports downstream clinical and research review
  • Available on Standard and Premium plans

AI Chatbot

  • Ask genetics and bioinformatics questions in context
  • Trained for variant interpretation and workflow guidance
  • Integrated directly into the platform UI

Run Analysis Pipeline

  • BWA-MEM2 service polls for pending FASTQ jobs

  • Sets status → VEP_ANALYZED

Import to MongoDB
Auto-triggered by cron every 30s — paths rejoin here
  • Find analyses with status VEP_ANALYZED → set IMPORTING
  • Run mongoimport from annotated TSV into a per-analysis collection
  • Create indexes: gene, CLNSIG, IMPACT, chrom+pos, rsId, PGx, and more
  • Count variants → set status ANALYZED, emit WebSocket update
  • Auto-generate PGx pharmacogenomics report
